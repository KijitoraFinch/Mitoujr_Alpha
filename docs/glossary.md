# Glossary

Artifact
: A unit that contains interpretable information.

Region
: A selectable part of an artifact.

Reference
: A value that targets an artifact or region.

Annotation
: Information attached to a region.

Relation
: A semantic connection between regions, references, or literal values.

Snapshot
: A recorded resolution result used for later comparison.

Diagnostic
: A stable report about invalid, inconsistent, or suspicious state.

Patch
: A proposed edit that can be applied by the core.

Derive
: Deterministically produce another explicit representation from existing
  explicit information.

Infer
: Produce candidate information that was not explicitly present.
